# Games Store

## Descrição

Projeto front-end de loja virtual de jogos desenvolvido como trabalho final para a disciplina de Desenvolvimento Web 1 do curso de Análise e Desenvolvimento de Sistemas do IFSP-CJO.

### Requisitos do projeto

- Mínimo de 5 páginas.
- Comentários no código-fonte.
- Lista ordenada e não ordenada.
- Formulário de no mínimo 5 campos.
- Tabela.
- Propriedades de formatação de fontes, textos, cores e fundos.
- 5 funcionalidades em JavaScript.

### Features

- Páginas dinâmicas.
- Layout responsivo.
- Filtro para gêneros de jogos e plataformas suportadas.
- Arquivo local JSON para armazenamento de dados de jogos.
- Persistência de dados na sessão via Local Storage.
- Validações para cadastro de usuário.

## Instalação

- Extraia o código fonte no diretório raiz do seu servidor local.
