# Games Store

## Description

Front-end project of a virtual game store developed as the final assignment for the Web Development 1 course of the Systems Analysis and Development program at IFSP-CJO.

### Project Requirements

- Minimum of 5 pages.
- Source code comments.
- Ordered and unordered lists.
- Form with at least 5 fields.
- Table.
- Font, text, color, and background formatting properties.
- 5 JavaScript functionalities.

### Features

- Dynamic pages.
- Responsive layout.
- Filter for game genres and supported platforms.
- Local JSON file for game data storage.
- Session Data persistence using Local Storage.
- User registration validations.

## Installation

- Extract the source code into the root directory of your local server.
